Everyday Ally is an AI-powered study planner and reminder assistant designed to help students stay organized, consistent, and productive. It generates personalized study plans, breaks tasks into manageable steps, adapts schedules automatically, and sends smart reminders so users never miss important sessions.  

Built using **Google AI Agent Tools**, the project focuses on making study planning simple, automatic, and stress-free for learners of all levels.
